<div style={{ padding: "20px", backgroundColor: "tomato" }}>
  <h3>This is JSX</h3>
</div>

## Getting Started

1. Customize `/public/index.html` as you need.
1. Add content to `/content/${folder}/index.mdx`
1. For dynamic content, add `/content/${folder}/index.js`

### TypeScript Support

1. Add https://github.com/TypeStrong/ts-node as a dependency.
1. Add `tsconfig.json`. (Run `tsc --init`)
1. Rename `.js` to `.ts` (or `.tsx`) as needed.
